# 📦 万物柜：面向高校的无人共享储物服务系统

万物柜是一款专为高校设计的**无人值守共享储物柜系统**，通过手机端小程序完成租用、管理与支付流程，提升校园内临时物品存取效率，解决高校快递、出行、临时物品寄存等场景中的空间调度问题。

---

## 🚀 核心功能

- 📱 **小程序界面交互**：扫码开柜、下单租用、一键归还、费用结算
- 🧠 **智能调度系统**：实时监测各柜使用情况，实现空间资源最优分配
- 🔐 **安全保障机制**：柜门自动锁闭，异常状态上传后台，支持人脸识别或动态密码验证
- 💰 **价格模型与结算机制**：按小时计费并动态调整，支持月卡/次卡形式

---

## 🧩 设计亮点

- 多设备协同控制：支持主柜、副柜无线联动
- 模块化硬件架构：便于不同校园规模部署
- 云端后台管理平台：查看订单、维护状态、营收报表一站式完成
- 低功耗待机 + 高频响应机制：兼顾成本与性能

---

## 🔧 技术实现概览

- 前端：微信小程序 + Web 后台管理界面
- 后端：Spring Boot + MySQL + Redis 缓存
- 通讯协议：MQTT 与蓝牙结合实现设备控制与状态同步
- 硬件：STM32主控 + 电磁锁 + 超声波传感 + OLED显示屏 + 4G模组

---

## 🌐 应用场景

- 📦 快递临时寄存
- 🎒 图书馆/自习室临时存包
- 🎮 赛事/活动临时装备存放
- 🛍️ 校园内外自助物品交接
